# House Price Prediction using Machine Learning

This project predicts house prices using Machine Learning algorithms such as Linear Regression, Random Forest Regressor, and Support Vector Regression (SVR).

## Features
- Data Cleaning and Preprocessing
- One-Hot Encoding
- Feature Scaling
- Correlation Heatmap Visualization
- Linear Regression Model
- Random Forest Regressor
- Support Vector Regression (SVR)
- Model Evaluation

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Jupyter Notebook

## Dataset
House Prices Dataset from Kaggle.

## Results
Important features affecting house prices include:
- Total Basement Area
- Year Built
- Year Remodeled
- Lot Area

Random Forest and SVR performed better on complex relationships, while Linear Regression provided a strong baseline model.
